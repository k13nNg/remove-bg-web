About

The project's main purpose is to remove background from users' uploaded images, using Python's OpenCV AI module.

The website's back-end operates mainly on Python, Javascript, and Django.

The website's front-end was created using: HTML, CSS, and Javascript.

The Image Processing Module was created by implementing the rembg module. More information could be found here.

This project was made possible by Kien, Ray, and Michael; who are currently students at the University of Waterloo.
